# المساعد الذكي - اتحاد طلاب تحيا مصر

## وصف المشروع

المساعد الذكي لاتحاد طلاب تحيا مصر - تطبيق محادثة ذكي يساعد في الإجابة على الأسئلة المتعلقة بالكيان الشبابي.

## التقنيات المستخدمة

هذا المشروع مبني باستخدام:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## كيفية تشغيل المشروع محلياً

```sh
# تثبيت المتطلبات
npm install

# تشغيل الخادم المحلي
npm run dev
```

## البناء للإنتاج

```sh
npm run build
```
